Project 01, Prized Possessions
What’s the most prized possession that you have in your apartment? Why do you treasure it? How did you acquire the item? What’s its intrinsic value? Based on the content that we collectively gather in this session, every week, you have to create 3 unique webpages using the technical proficiency that you acquire during the class.
By the end of week 4, I expect you all to have atleast 12 unique webpages.

- Introduce basic HTML tags (<p></p>, <br>, <hr>, and &nbsp;)

Next week?
- Read Laurel Schwulst, My Website is a Shifting House...
- Create 3 unique webpages only using those tags (
due by Jan. 27th
) based on the content that we collectively gather in the class
